# leaflet-player
A player to present geojson in a time series

##Denpendencies
- [leaflet](https://github.com/Leaflet/Leaflet)
- [vis.js](https://github.com/almende/vis)
- [q.js](https://github.com/kriskowal/q)
- [jquery](https://github.com/jquery/jquery)

##Development
